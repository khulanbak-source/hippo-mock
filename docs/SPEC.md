# Hippo Mock Exam — Design & Exam-Format Notes

## Goal
A phone/iPad-friendly, timed practice site for a 10-year-old preparing for the **Hippo English
Olympiad — Little Hippo category, Continental (Qualifier) round**. Deployed at
`mylittletest.urstory.studio`.

## Real exam format (from the official Hippo 2025 Regulations)
- **Little Hippo** = school grades 3–4. CEFR **Pre-A1 → A1 Young Learners**.
- **Continental / Qualifier round** = **30 open-ended Use-of-English questions + 1 crossword.**
  No reading test, no essay.
- **No negative marking.** One correct answer per question; where more than one answer is
  correct, all are accepted.
- **Timing:** the regulations publish times for the Preliminary and Final rounds but **not** the
  Qualifier round. The Little Hippo preliminary sitting is Reading 20 min + Use of English 20 min
  (= 40 min). We replicate **40 minutes** for the one-crossword + 30-question sitting.
- **Scoring:** Hippo ranks on the **average of the section percentages**. We mirror this:
  `overall = average(crossword%, useOfEnglish%)`, **pass ≥ 75%**.

## App flow
1. **Login** — Name + Code, validated live against the Notion *Users* DB via `api/login.js`.
2. **Ready** — rules + saved progress ("⭐ Passed: N / 10"). Tap **Start exam**.
3. **Exam** — 40:00 countdown starts. Crossword + 30 fill-in questions on one page.
   - Timer turns amber ≤ 5 min, red ≤ 60 s; at 0:00 it auto-submits.
   - Refresh / Start over loads a fresh shuffled exam.
4. **Result**
   - **Pass (≥75%)**: a **See result** button gates the score reveal (reward).
   - **Fail (<75%)**: score shows immediately with **Try again**.
   - Either way, **Review my answers** lists every item with the correct answer.

## Content
- **10 exams**, each = crossword (theme) + 30-question Use-of-English test, paired by index.
- **Shuffled**: each Start picks a not-yet-passed exam at random; when all 10 pass → "ask for 10 more."
- Crosswords are generated by a greedy interlock builder (`build_data.py`) → guaranteed valid grids.
- Use-of-English items are open-ended (type the word). A `(clue)` in brackets forces a single
  answer; `answers[]` lists every accepted variant. Auto-graded case-insensitively, punctuation
  ignored, blanks not penalised.

## Architecture
- Static site (`index.html` + `styles.css` + `app.js` + generated `data.js`), **no build step**.
- One serverless function `api/login.js` (Notion validation). `NOTION_TOKEN` lives only in Vercel
  env, never in git.
- Grading is client-side (answers ship in `data.js`). Acceptable for a kid's practice tool.

## Decisions
- Open-ended (not MCQ) to match the real Continental round.
- 40-minute single timer (no official qualifier time published; mirrors Little Hippo sitting).
- Average-of-two-sections scoring, 75% pass, no negative marking (mirrors Hippo).
- Live Notion auth so users are managed in Notion, not in code.
